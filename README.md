# 🌐 Networking-Labs

## 💻 About this repository

This repository contains my practical networking labs developed as part of my **ASIR (Systems Administration)** studies.

It is focused on building real understanding of how networks work through simulation tools such as **Cisco Packet Tracer**, along with basic network configuration, troubleshooting, and analysis.

The main goal is to develop practical networking skills by designing, configuring, and testing different network scenarios in controlled environments.

---

## 🧠 What this repository is about

This repository is focused on practical networking concepts such as:

- Network design and topology creation
- IP addressing and subnetting
- Routing between networks
- VLAN configuration and segmentation
- Network services (DHCP, DNS, HTTP simulation)
- Network security basics (ACLs and access control)
- Troubleshooting and diagnostics

Each topic is explored through hands-on labs and real simulations.

## 🎯 Purpose

The purpose of this repository is to:

- Understand how real computer networks are structured
- Practice designing and configuring network topologies
- Learn how devices communicate across networks
- Develop troubleshooting and diagnostic skills
- Apply theoretical networking knowledge in practical scenarios
- Strengthen skills required for systems administration and networking roles

---

## 📈 Learning approach

My learning process is based on practical simulation and experimentation:

- Building network topologies from scratch
- Configuring routers, switches, and end devices
- Testing connectivity between systems
- Simulating real-world network scenarios
- Identifying and solving configuration errors
- Documenting each step of the process

Most practical exercises in this repository are developed using **Cisco Packet Tracer**, allowing the simulation of real network environments such as:

- Educational institution networks
- Multi-subnet environments
- VLAN-separated infrastructures
- Routed networks with multiple segments
